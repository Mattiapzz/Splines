class Spline2D : public handle¶

MATLAB class wrapper for the underlying C++ class.

The construction of the 2D spline is done as follows

  • instantiate the spline object

spl = Spline2D( kind, X, Y, ZZ );
% or
spl = Spline2D( kind );
spl.build( X, Y, ZZ );

kind is a string and can be any of

kind

meaning

‘bilinear’

piecewise linear in X e Y direction

‘cubic’

piecewise cubic in X e Y direction

‘akima’

piecewise cubic with Akima non oscillatory construction

‘biquintic’

piecewise quintic in X e Y direction

Evaluation is simple

Z = spl.eval( X, Y );

where X and Y are scalar or vector or matrix of the same size. the result Z is scalar or vector or matrix of the same size of the inputs.

example of usage

X = -2:0.01:2;
Y = -2:0.01:2;
[XX,YY] = ndgrid(X,Y);
ZZ  = peaks(XX,YY);
spl = Spline2D('bicubic',X,Y,ZZ);

surf(XX,YY,ZZ), view(145,-2), set(gca,'Fontsize',16);

X = -2:0.1:2;
Y = -2:0.1:2;
[XX,YY] = ndgrid(X,Y);

ZZ = spl.eval(XX,YY);

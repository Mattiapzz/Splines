function SplineSet( varargin)¶

Build a spline Set

X = [ .... ];                  % vector or nodes
Y = [ .... ];                  % matrix of values or
Y = { [....],  ...., [....] }; % cell array of vectors
kind = { .... }                % cell array of strings
                               % with the kind of each spline

obj = SplineSet( kind, X, Y );

Kind is a string and can be any of

kind

meaning

‘linear’

linear spline (only continuous)

‘cubic’

cubic spline ( \( C^2 \) function)

‘akima’

Akima non oscillatory spline ( \( C^1 \) function)

‘bessel’

bessel non oscillatory spline ( \( C^1 \) function)

‘pchip’

Monotone \( C^1 \) function

‘hermite’

Hermite spline (set \( p_k \) and \( p'_k \))

‘quintic’

Quintic spline ( \( C^3 \) function)
